Task 3 - API Integration

Project: Fetches public API data and displays user cards with search and refresh support.

Objective

This task shows how to fetch external data using JavaScript and present it with loading/error states.

Implemented Features

  • Fetch from JSONPlaceholder API
  • Live search filtering by name, username, and city
  • Refresh button to reload data
  • Loading and error states

Code Snippet

async function loadApiData() {
  setApiStatus('Loading data...');
  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/users');
    apiUsers = await response.json();
    filterApiUsers();
  } catch (error) {
    setApiStatus('Unable to fetch API data.', true);
  }
}
