Task 2 - Mini Interactive JavaScript App

Project: To-Do app with add, complete, filter, and clear completed features.

Objective

This task demonstrates DOM manipulation and localStorage-based state handling in JavaScript.

Implemented Features

  • Task creation and completion toggling
  • Filtering between all/active/completed tasks
  • Local storage persistence
  • Clear completed action

Code Snippet

function addTask(text) {
  const trimmed = text.trim();
  if (!trimmed) return;
  tasks.unshift({ id: Date.now().toString(), text: trimmed, completed: false });
  saveTasks();
  renderTasks();
}
