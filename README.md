# My_portfolio
# Portfolio Project Structure

This repository contains a responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- Responsive portfolio layout with sections for Home, About, Skills, Projects, API, and Contact
- Sticky navigation with smooth scrolling and mobile menu support
- Interactive To-Do app with add, complete, filter, and clear actions
- Public API integration using Fetch with loading, error, and search states
- Netflix-style UI clone section for a polished frontend challenge

## Project Structure
- `index.html` - Main portfolio page
- `css/style.css` - Main stylesheet
- `js/script.js` - Interactive behavior and API logic
- `task_docs/` - Submission-ready task documentation pages

## How to Run
Open `index.html` in your browser, or serve the project locally using a simple static server.

Example:
```bash
python -m http.server 8000
```
Then open: http://127.0.0.1:8000/

## Technologies Used
- HTML5
- CSS3
- JavaScript ES6+
- ScrollReveal for animations

## Deployment
This project is ready to be deployed on GitHub Pages, Netlify, or Vercel.
